# 03 【 CSS技巧 】

# 1. 繪製三角形

<body>
    <div class="box"></div>
</body>

.box{
    width: 0;
    height: 0;
    border-top: 250px solid transparent;
    border-right: 250px solid transparent;
    border-bottom: 250px solid #000;
    border-left: 250px solid transparent;
}
.box{   /* 精簡寫法 */
    width: 0;
    height: 0;
    border: 250px solid transparent;
    border-bottom-color: brown;
}

# 2. 繪製圓形

<body>
<div class="box"></div>
</body>

.box{
    width: 500px;
    height: 500px;
    border: 1px solid #000;
    border-radius: 250px; /* 大小為長寬的一半 */
    background-color: #000;
}
.box{
    width: 500px;
    height: 500px;
    border: 1px solid #000;
    border-radius: 50%; /* 百分比為長寬的一半 */
    background-color: #000;
}

# 3. 元素水平居中

# 1. text-align: center

<body>
<h1>h1標籤</h1>
<p>p標籤</p>
<div>
    <span>span標籤</span>
</div>
</body>

span為行內元素，必須在行內元素的父元素加上CSS樣式才會生效

h1{
    text-align: center;
}
p{
    text-align: center;
}
div{ /* div為span標籤的父元素 */
    text-align: center;
}

# div標籤

div元素本身不會水平置中，但會對標籤內的文字、元素生效

若要讓div元素水平置中，需要使用margin: 0 auto

<body>
<!-- 文字會水平置中 -->
<div>
    div標籤
</div>
</body> 

div{
    width: 500px;
    height: 20px;
    text-align: center;
    background-color: blanchedalmond;
}

# 2. margin: 0 auto

只對區塊元素生效

<body>
<div></div>
</body>

div{
    width: 200px;
    height: 200px;
    background-color: #000;
    margin: 0 auto;
}

# 只有div需要水平居中

<body>
    <h1>靠左</h1>
    <div></div>
    <h1>靠左</h1>
</body>

div{
    width: 200px;
    height: 200px;
    background-color: #000;
    margin: 0 auto;
}
h1{
    text-align: left;
}

# 3. 浮動元素

<body>
    <div class="box"></div>
    <div class="line"></div>
</body>

.box{
    width: 200px;
    height: 200px;
    background-color: #000;
    float: left;
    position: relative;
    left: 50%;
    
    /* 相同效果 */
    margin-left: -100px; /* 向左偏移元素大小的一半 */
    transform: translateX(-50%);
}
.line{
    width: 5px;
    height: 100vh;
    margin: 0 auto;
    background-color: #000;
}

# 4. 絕對定位的行內元素

# 1. 在行內元素的父元素上設置text-align: center

<body>
<span>span標籤</span>
<div class="line"></div>
</body>

body{
    text-align: center;
}
span{
    color: brown;
    position: absolute;
    left: 0;
    right: 0;
}
.line{
    width: 5px;
    height: 100vh;
    margin: 0 auto;
    background-color: #000;
}

# 2. left: 50% + transform: translateX(-50%)

<body>
<span>span標籤</span>
<div class="line"></div>
</body>

body{
    text-align: center;
}
span{
    color: brown;
    position: absolute;
    left: 0;
    right: 0;
    left: 50%;
    transform: translateX(-50%)
}
.line{
    width: 5px;
    height: 100vh;
    margin: 0 auto;
    background-color: #000;
}

# 8. rem 與 em

# 1. rem

根元素的font-size

<body>
    <div class="div1">div1</div>
    <div class="div2">div2</div>
    <div class="div3">div3</div>
    <div class="div4">div4</div>
</body>

html{
    color: red;
    font-size: 1rem;    /* 瀏覽器預設大小為16px */
}
.div1{
    font-size: 1rem;
}
.div2{
    font-size: 2rem;
}
.div3{
    font-size: 3rem;
}
.div4{
    font-size: 64px;
}

# 2. em

先查找元素自身是否有設定font-size，再查找父元素的font-size

<body>
    <div class="div1">
        <div class="div2"></div>
        <div class="div3"></div>
    </div>
</body>

.div1{
    background-color: lightblue;
    font-size: 20px;
    height: 10em;
    width: 10em;
}
.div2{
    background-color: lightpink;
    width: 5em;
    height: 5em;
}
/* 結果相同，以元素本身設定的font-size來決定大小 */
/* .div2{
    background-color: lightpink;
    font-size: 10px;
    width: 10em;
    height: 10em;
} */
.div3{
    background-color: lightgreen;
    width: 100px;
    height: 100px;
}

# 9. 偽類 與 偽元素

語法數量位置類與元素使用場景
偽類:多個前方與後方修飾多
偽元素::單個後方建立對象少

# 1. 偽類

  1. 狀態類
:link:visited:hover:active:focus
  1. 結構類
:first-child:last-child:nth-child:nth-of-type
  1. 表單類
:checked:disabled:valid:required
  1. 語言類
:dir:lang

# 範例

# 1. 偽類拼接

數字超出範圍後改變背景顏色並顯示於結果

<body>
<h1>請輸入數字</h1>
<input type="number" min="1" max="100">
<h2>結果：</h2>
</body>

input{
    font-size: 3em;
    width: 200px;
    height: 50px;
}
input:out-of-range:focus{background-color: burlywood;}
input:out-of-range + h2::after{content: "超出範圍";}
# 2. 勾選時出現已勾選
<body>
    <input type="checkbox">
    <input type="checkbox">
    <input type="checkbox">
</body>

input{
    width: 100px;
    height: 100px;
}
input::after{
    content: "已勾選";
    display: none;
    width: 150px;
    height: 50px;
}
input:checked::after{
    display: block;
}

# 2. 偽元素

::before::after::first-letter::first-line::selection::placeholder::backdrop

# 範例

偽類是基於DOM產生不同的狀態，偽元素是建立一個不存在DOM裡的新對象

<body>
<ol>
    <li>第一個</li>
    <li>第二個</li>
    <li>第三個</li>
    <li>第四個</li>
</ol>
<p>一二三四</p>
</body>

li:nth-child(2n+1){
    color: red;
}
/* 偽元素不會偽元素不會 */
p::first-letter{
    color:red;
}

# 10. 垂直居中

# 1. padding

優點：只須設定上下內邊距

缺點：父元素不能設定高度

<body>
    <div>
        <span>文字</span>
    </div>
</body>

div{
    width: 200px;
    padding-top: 50px;
    padding-bottom: 50px;
}
span{
    font-size: 50px;
}

# 2. line-height

優點：可設定父元素高度

缺點：只適合用於單行文字

<body>
    <div>
        <p>文字</p>
    </div>
</body>

div{
    width: 350x;
    height: 100px;
}
p{
    font-size: 50px;
    line-height: 100px;
}

# 3. Flexbox

優點：簡單易懂

缺點：IE9以上兼容性

<body>
    <div>
        <a href="">文字</a>
    </div>
</body>

div{
    height: 210px;
    width: 170px;
    display: flex;
    align-items: center;
    justify-content: center;
}

# 4. table

缺點：一般用於表格

<body>
    <ul>
        <li><a href="">一</a></li>
        <li><a href="">二</a></li>
        <li><a href="">三</a></li>
        <li><a href="">四</a></li>
        <li><a href="">五</a></li>
    </ul>
</body>

ul{
    display: table;
}
li{
    display: table-cell;
    vertical-align: middle;
}

# 5. grid

優點：適合用來作布局

缺點：兼容性，難度高

<body>
    <ul>
        <li><a href="">一</a></li>
        <li><a href="">二</a></li>
        <li><a href="">三</a></li>
        <li><a href="">四</a></li>
        <li><a href="">五</a></li>
    </ul>
</body>

ul{
    display: table;
}
li{
    display: table-cell;
    vertical-align: middle;
}

# 6. position

優點：難度低

缺點：脫離排版

<body>
    <div>
        <span>文字</span>
    </div>
</body>

div{
    position: relative;
    width: 200px;
    height: 200px;
}
span{
    height: 50px;
    width: 50px;
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
}

# 7. 偽元素

優點：兼容性佳

缺點：不易理解

<body>
    <div class="main">
        <div class="search">
            <form action="">
                <input type="text" placeholder="搜尋">
                <div class="button"><i></i></div>
            </form>
        </div>
    </div>
</body>

div.search{
    display: inline-block;
    vertical-align: middle;
}
div.main::after{
    content: "";
    display: inline-block;
    background-color: bisque;
    height: 100%;
    width: 0;
    vertical-align: middle;
}
