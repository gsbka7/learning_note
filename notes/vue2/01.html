# 01 【插值】

# 1. Mastache 語法

使用 {{}} 渲染資料

<template>
  <div>
    {{name}}
    {{age}}
  </div>
</template>

<script>
export default {
  data(){
    return {
      name: 'user',
      age: 20
    }
  }
}
</script>

# 2. v-once

被定義 v-once 的元素或組件(包含元素或組件內的所有子節點)只能被渲染一次。

即使資料發生變化，也不會被重新渲染。一般用於靜態內容展示。

<template>
  <div>
      <h2>{{message}}</h2>
      <h2 v-once>{{message}}</h2>
  </div>
</template>

<script>
export default {
  data(){
    return {
      message: 'hello'
    }
  }
}
</script>

# 3. v-html

使用 {{}} 會顯示

使用 v-html 會渲染HTML標籤

<template>
<div id="app">
  <div>{{ message }}</div>
  <div v-html="message"></div>
<!--    -->
</div>
</template>

<script>
export default {
  data(){
    return {
       message: '<h1> Hello! </h1>'
    }
  }
}
</script>

# 4. v-text

與 {{}} 相同

<template>
<div id="app">
      <div>{{message}}</div>
      <div v-text="message"></div>
</div>
</template>

<script>
export default {
  data(){
    return {
        message: 'hello'
    }
  }
}
</script>

# 5. v-pre

使用 {{}} 會顯示 hello

使用 v-pre 會顯示 

<template>
<div id="app">
      <div>{{message}}</div>
      <div v-pre>{{message}}</div>
</div>
</template>

<script>
export default {
  data(){
    return {
        message: 'hello'
    }
  }
}
</script>

# 5. v-cloak

進入頁面時，因為網路速度緩慢，內容尚未解析完成就會被渲染至畫面上(如：{{data}})

使用 v-cloak 可以隱藏未解析的內容

<template>
<div id="app" v-cloak>
      <div>{{message}}</div>
</div>
</template>

<script>
export default {
  data(){
    return {
        message: 'hello'
    }
  }
}
</script>

<style>
    [v-cloak]{
        display: none;
    }
</style>
