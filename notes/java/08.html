# 08 【字串 String】

# 1. 概述

java.lang.String類代表字串，java程式中所有字串文字(例如："abc")都為此類

String name = "名字";
String age = "年齡";

# 注意點：

字串內容是不會發生改變的，物件在建立後不能被修改

String name = "名字";
name = "Name";

並不是修改原來字串的內容，而是建立了一個新的字串

# 總結：

  1. String是java定義好的一個類。定義在java.lang包中，所以使用時不需要導入

  2. java程式中的所有字串文字都被歸為此類的物件

  3. 字串的值在建立後不能被修改

# 2. 建立String物件的方式

直接賦值 與 new

        // 1. 直接賦值
        String s1 = "abc";
        System.out.println(s1); // abc


        // 2. new
        // 空參構造：可以取得一個空白的字串
        String s2 = new String();
        System.out.println("@"+s2+"!"); // @!

        // 傳遞一個字串，根據傳遞的字串內容在建立一個新的字串
        String s3 = new String("abc");

        // 傳遞一個字串陣列，根據字串陣列的內容在建立一個新的字串
        // 可用於修改字串的內容
        // abc --> {'a','b','c'} --> {'z','b','c'}; --> zbc
        char[] chs = {'a','b','c','d'};
        String s4 = new String(chs);
        System.out.println(s4); // abcd

        // 傳遞一個byte陣列，根據陣列的內容建立新的字串
        // 可用於將byte訊息轉換成字串
        byte[] bytes = {97,98,99,100};
        String s5 = new String(bytes);
        System.out.println(s5); // abcd

# 3. 字串比較

# 1. 基本資料類型 與 參考資料類型

基本資料類型：比較的是資料的值

int a = 10;
int b = 20;
System.out.println(a == b); // false

參考資料類型：比較的是位址

String s1 = new String("abc");
String s2 = new String("abc");
System.out.println(s1 == s2); // false

# 2. 字串比較

String s1 = "abc";
String s2 = "abc"; // 直接賦值的會在串池，因為兩個值相等所以會複用
System.out.println(s1 == s2); // true

String s1 = new String("abc"); // 記錄堆的位址
String s2 = "abc"; // 記錄串池的位址
System.out.println(s1 == s2); // false

# 字串比較的方法

boolean equals()：完全一樣的結果才是true，否則為false(區分大小寫)

boolean equalsslanoreCase()：忽略大小寫

# 4. StringBuilder

StringBuilder可以看成是一個容器，建立之後裡面的內容是可變的

# 1. 構造方法

public StringBuilder()：建立一個空白可變字串物件，不含有任何內容

public StringBuilder(String str)：根據字串的內容，來建立可變字串物件

# 2. 常用方法

public StringBuilder appedn(任意類型)：增加資料，並返回物件本身

public StringBuilder reverse()：反轉容器中的內容

public int lenght()：返回長度

public String to String()：通過toString()可以實現把StringBuilder轉換成String

public class StringBuilderDemo01 {
    public static void main(String[] args) {
         // 1. 建立物件
        StringBuilder sb = new StringBuilder("abc");

        // 2. 增加元素
        sb.append(1); // abc1
        sb.append(2.3); // abc12.3
        sb.append(true); // abc12.3true

        // 3. 反轉
        sb.reverse(); // eurt3.21cba

        // 4. 取得長度
        int len = sb.length(); // 11

        System.out.println(sb);
        System.out.println(len);
    }
}

public class StringBuilderDemo02 {
    public static void main(String[] args) {
         StringBuilder sb = new StringBuilder();

         sb.append("aaa");
         sb.append("bbb");
         sb.append("ccc");
         sb.append("ddd");

        // StringBuilder只是一個操作字串的容器，並不是字串
        System.out.println(sb);

        // 把StringBuilder變回字串
        sb.toString();
        System.out.println(sb);
    }
}

# 範例：
  1. 檢查輸入的字串是否對秤
import java.util.Scanner;

public class StringBuilderDemo02 {
    public static void main(String[] args) {
        
        Scanner sc = new Scanner(System.in);
        System.out.println("請輸入一個字串");
        String str = sc.next();

        StringBuilder sb = new StringBuilder();
        sb.append(str);
        sb.reverse();
        System.out.println(sb);

        String result = new StringBuilder().append(str).reverse().toString();
        if(str.equals(result)) {
            System.out.println("對稱");
        }else{
            System.out.println("不對稱");
        }
    }
}
  1. 拼接字串
public class StringBuilderDemo02 {
    public static void main(String[] args) {

        int[] arr = {1, 2, 3};

        String str = arrToString(arr);
        System.out.println(str);
    }

    public static String arrToString(int[] arr) {
        StringBuilder sb = new StringBuilder();
        sb.append("[");

        for (int i = 0; i < arr.length; i++) {
            if (i == arr.length - 1) {
                sb.append(arr[i]);
            } else {
                sb.append(arr[i] + ", ");
            }
        }
        sb.append("]");
        return sb.toString(); // StringBuilder只是容器，要返回toString
    }
}

# 5. StringJoiner

StringJoiner與StringBuilder一樣，也可以看成是一個容器，建立之後裡面的內容是可變的

用於提高字串的操作效率，而且程式碼編寫特別簡潔

JDK8出現的

# 1. 構造方法

public StringJoiner(間隔符號)：建立一個StringJoiner物件，指定拼接的間隔符號

public StringJoiner(間隔符號, 開始符號, 結束符號)：建立一個StringJoiner物件，指定拼接的間隔符號、開始符號、結束符號

StringJoiner sj = new StringJoiner("---") // 1---2---3
StringJoiner sj = new StringJoiner(", ","[","]") // 1---2---3

# 2. 成員方法

public StringJoiner add(增加的內容)：增加資料，並返回物件本身

public int length()：返回長度

public String toString()：返回一個字串(該字串就是拼接之後的結果)

import java.util.StringJoiner;

public class StringJoinerDome01 {
    public static void main(String[] args) {
        StringJoiner sj = new StringJoiner("---");

        sj.add("aaa").add("bbb").add("ccc");

        System.out.println(sj); // aaa---bbb---ccc
    }
}

import java.util.StringJoiner;

public class StringJoinerDome02 {
    public static void main(String[] args) {
        StringJoiner sj = new StringJoiner(", ","[","]");

        sj.add("aaa").add("bbb").add("ccc");
        int len = sj.length();
        System.out.println(len); // 15

        System.out.println(sj); // [aaa, bbb, ccc]

        // 轉換成字符
        String str = sj.toString();
        System.out.println(str); // [aaa, bbb, ccc]
    }
}

# 2. 總結

  1. String：

表示字串的類，定義了許多操作字串的方法

  1. StringBuilder

一個可變的操作字串的容器

可以有效率地拼接字串，還可以將裡面的內容反轉

  1. StringJoiner

JDK8出現的一個可辨的操作字串的容器，可以有效率、方便地拼接字串

# 6. 字串原理

  1. 字串儲存的記憶體原理

直接賦值會複用字串常數池中的資料

new出來的不會複用，而是開闢一個新的空間

  1. ==號比較的是什麼

基本資料類型比較的是資料值

參考資料類型比較的是位址值

  1. 字串拼接的原理

如果沒有變數參與，都是字串直接相加，編譯之後就是拼接完成的結果，會複用串池中的字串

如果有變數參與，每一行拼接的程式碼，都會在記憶體中建立新的字串，浪費記憶體

  1. StringBuilder提高效率原理

所有要拼接的內容都會往StringBuilder中存放，不會建立無用的空間，節省記憶體
