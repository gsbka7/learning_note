# 04 【陣列】

# 1. 定義

陣列是相同類型資料的有序組合

陣列描述的是相同類型的若干個資料，按照一定的先後次序排列組合而成

每一個資料稱為一個元素，每個元素可以通過索引來進行訪問

# 2. 陣列宣告建立

首先必須宣告陣列變數，才能在程式中使用陣列

dataType[] arrayRefVar; // 首選的方法

dataType arrayRefVar[]; // 效果相同，但不是首選方法

Java語言使用new操作符來建立陣列

dataType[] arrayRefVar = new dataType[arraySize];

陣列的元素是通過索引訪問的，陣列索引從0開始

取得陣列長度：array.length = 

public class Demo2 {
    public static void main(String[] args) {
        int[] nums; // 1.宣告一個陣列
        // int num2[];

        nums = new int[10]; // 2.建立一個陣列，可以存放10個int類型的數字

        // 3. 向陣列元素中賦值
        nums[0] = 1;
        nums[1] = 2;
        nums[2] = 3;
        nums[3] = 4;
        nums[4] = 5;
        nums[5] = 6;
        nums[6] = 7;
        nums[7] = 8;
        nums[8] = 9;
        nums[9] = 10;

        // 計算所有元素的和
        int sum = 0;
        for(int i = 0; i<nums.length; i++) {
            sum = sum+=nums[i];
        }
        System.out.println("總和為："+sum);
    }
}

# 3. 內部記憶體分析

# 堆：

存放new的物件和陣列

可以被所有的線程共享，不會存放別的物件引用

# 棧：

存放基本變數類型(會包含這個基本類型的具體數值)

引用物件的變數(會存放這個引用在堆裡面的具體地址)

陣列在宣告時並不存在，到了建立陣列時才存在

一般會把宣告跟建立寫在一起

public class Demo2 {
    public static void main(String[] args) {
        int[] nums; // 1.宣告一個陣列   

        nums = new int[10]; // 2.建立一個陣列，可以存放10個int類型的數字;
    }
}

# 陣列的三種初始化狀態

# 靜態初始化

手動指定陣列長度，由系統給出默認初始值

int[] a = {1,2,3};
Man[] mans = {new Man(1,1), new Man(2,2)};

# 動態初始化

手動指定陣列元素，系統會根據元素個數，算出陣列的長度

int[] a = new int[2];
a[0]=1;
a[1]=2;

# 陣列的默認初始化

陣列是引用類型，它的元素相當於類的實例變數，因此陣列一經分配空間，其中的每個元素也被按照實例變數同樣的方式被隱式初始化

陣列默認初始化的規律

整數：0

小數：0.0

字符：/u0000(空格)

布爾：false

參考資料型態：null

public class Demo2 {
    public static void main(String[] args) {
        // 靜態初始化：建立 + 賦值
        int[] a= {1,2,3,4};
        System.out.println(a[0]);

        // 動態初始化：包含默認初始化
        int[] b = new int[10];
        b[0] = 10;
        b[1] = 0;
        System.out.println(b[0]);

        // 默認初始化：沒有被初始化的都是0
        System.out.println(b[1]);
        System.out.println(b[2]);
        System.out.println(b[3]);
    }
}

# 4. 陣列的四個基本特點

其長度是確定的。陣列一旦被建立，它的大小就是不可以改變的

其元素必須是相同類型，不允許出現混和類型

陣列的元素可以是任何資料類型，包括基本類型和引用類型

陣列變數屬於引用類型，陣列也可以看成是物件，陣列中的每個元素相當於該物件的成員變數。陣列本身就是物件，Java中物件是在堆中的(使用new建立的都會在堆中)，因此陣列無論保存原始類型還是物件類建，陣列物件本身是在堆中的

# 5. 陣列邊界

索引的合法區間：[0, length-1]，如果超過就會報錯

public class Demo3 {
    public static void main(String[] args) {
        int[] a = new int[2];
        System.out.println(a[2]);
        
        // ArrayIndexOutOfBoundsException：陣列索引異常
    }
}

常見錯誤範例：

i < a.length; 與 i <= a.length;

public class Demo3 {
    public static void main(String[] args) {
        int[] a = {1, 2, 3, 4, 5, 6, 7, 8};
        for(int i = 0; i < a.length; i++) {
            // 輸出 1 ~ 8
            System.out.println(a[i]);
        }

        for(int i = 0; i <= a.length; i++) {
            // 輸出 1 ~ 8 + ArrayIndexOutOfBoundsException
            System.out.println(a[i]);

            /* 
                a.length = 8
                當i = 8會輸出a[8];
                但是陣列索引只到7
                從 0 開始 7 結束，8 就超過了
             */
        }
    }
}

# 小結

陣列是相同資料類型(資料類型可以為任意類型)的有序集合

陣列也是物件。陣列元素相當於物件的成員變數

陣列長度是確定的，不可變的。如果越界，則會出現ArrayIndexOutOfBoundsException

# 6. 陣列使用

# 基本使用

public class Demo3 {
    public static void main(String[] args) {
        int[] arrays = {1,2,3,4,5};

        // 印出所有的陣列元素
        for(int i = 0; i < arrays.length; i++) {
            System.out.println(arrays[i]);
        }

        // 計算元素的和
        int sum = 0;
        for(int i = 0; i < arrays.length; i++) {
            // sum = sum + arrays[i];
            sum += arrays[i];
        }
        System.out.println("所有元素的和："+sum);

        // 查找最大元素
        int max = arrays[0];

        for(int i = 1; i < arrays.length; i++) {
            if(arrays[i] > max) { // 如果max大於循環的元素
                max = arrays[i]; // 將該元素賦值給max
            }
        }
        System.out.println("最大值："+max);
    }
}

# 4. 陣列作為返回值

public class Demo1 {
    public static void main(String[] args) {
        int[] arrays = {1,2,3,4,5};

        int[] reverse = reverse((arrays)); // int[] reverse是已經被反轉過的陣列
        printArray(reverse); // 輸出所有元素
    }

    // 輸出所有元素
    public  static void  printArray(int[] arrays) { // 陣列封裝程參數
        for (int i = 0; i < arrays.length; i++) {
            System.out.println(arrays[i] + "");
        }
    }

    // 反轉陣列
    public static int[] reverse(int[] arrays) {
        int[] result = new int[arrays.length];

        for (int i = 0, j = result.length-1; i < arrays.length; i++, j--) {
            // 把array的第一個元素賦值給result的最後一個元素
            result[j] = arrays[i];
        }
        return result;
    }
}

# 8. Arrars類

陣列的工具類java.util.Arrays

由於陣列物件本身並沒有什麼方法可以提供呼叫，但API中提供了一個工具類Arrays，對資料物件進行一些基本的操作

Array類中的方法都是static修飾的靜態方法，在使用的時候，可以直接使用類名進行呼叫，而"不用"使用物件來呼叫

# 具有以下常用功能：

給陣列賦值：通過fill方法

對陣列排序：通過sort方法，升序

比較陣列，通過equals方法比較陣列中元素是否相等

查找陣列元素，通過binarySearch方法能對排序好的陣列進行二分查找法操作

import java.util.Arrays; // 導入工具類

public class Array {
    public static void main(String[] args) {
        int[] a = {1,2,3,4,999,34,4645,6,4325,435,436};

        // 印出陣列元素
        System.out.println(Arrays.toString(a)); // [1, 2, 3, 4, 999, 34, 4645, 6, 4325, 435, 436]

        // 陣列元素升序
        Arrays.sort(a);
        System.out.println(Arrays.toString(a)); // [1, 2, 3, 4, 6, 34, 435, 436, 999, 4325, 4645]

        printArray(a); // [1, 2, 3, 4, 6, 34, 435, 436, 999, 4325, 4645]


        // 陣列填充：將指定的直噴配給陣列裡的每個元素
        Arrays.fill(a, 0); // 全部元素替換為0
        Arrays.fill(a,2,4,0); // 索引的2到4，左閉右開取值，元素3跟4會被替換為0 // [1, 2, 0, 0, 6, 34, 435, 436, 999, 4325, 4645]
        System.out.println(Arrays.toString(a));

    }

    // 手寫Arrays.toString方法
    public static void printArray(int[] a) {
        for (int i = 0; i < a.length; i++) {
            if(i==0) {
                System.out.print("[");
            };
            if(i==a.length-1) {
                System.out.print(a[i]+"]");
            }else{
            System.out.print(a[i]+", ");
            }
        }
    }
}

# 9. 冒泡排序

冒泡排序是最出名的排序算法之一，總共有八大排序


public class Array {
    public static void main(String[] args) {
        int[] a = {1,254,3,534};

        int[] sort = sort(a); // 呼叫後會產生排序後的陣列
        System.out.println(Arrays.toString(sort));
    }

    // 冒泡排序
    // 1. 比較陣列中，兩個相鄰的元素，如果第二個值比第一個大，就交換位置
    // 2. 每一次比較，都會產生一個最大，或最小的數字
    // 3. 下一輪則可以少一次排序
    // 4. 一次循環，值到結束

    public static int[] sort(int[] array) {
        // 臨時變數
        int temp = 0;

        // 外層循環，判斷要走多少次
        for (int i = 0; i < array.length - 1; i++) {
            // 內層循環：比較判斷兩個數，如果第二個值比第一個值大，就交換位置

            boolean flag = false; // 減少沒有意義的比較

            for (int j = 0; j < array.length - 1 - i; j++) { // 減去的i為已經比較
                if (array[j + 1] < array[j]) {
                    temp = array[j];
                    array[j] = array[j + 1];
                    array[j + 1] = temp;
                    flag = true;
                    /*
                    如果需要排序，flag會改變為true，循環完後再回到上面改變為false。
                    如果不需要排序，flag的值不會改變，直接跳至if語句然後跳出程式
                     */
                }
                if(flag==false) {
                    break;
                }
            }
        }
        return array;
    }
}

# 10. 稀疏陣列

當一個陣列中的大部分元素為0，或者為同一值的陣列時，可以使用稀疏陣列來保存該陣列

稀疏陣列的處理方式：

  1. 紀錄陣列一共有幾行幾列，有多少個不同值

  2. 把具有不同值的元素或行列及值記錄在一個小規模的陣列中，從而縮小程式的規模

# 稀疏陣列圖解

[0]：6行、7列、8個值

[1]：橫坐標、縱坐標、值

# 範例：

public class Array {
    public static void main(String[] args) {
        // 1. 建立一個二維陣列 11*11    0：沒有棋子，1：黑棋，2：白棋
        int[][] array1 = new int[11][11]; // 11行 11列 // 沒有賦值，默認輸出0
        array1[1][2] = 1;
        array1[2][3] = 2;

        // 輸出原始的陣列
        System.out.println("輸出原始的陣列");

        for (int[] ints : array1) {
            for (int anInt : ints) {
                System.out.print(anInt+"\t");
            }
            System.out.println();
        }

        // 轉換為稀疏陣列保存
        // 取得有效值的個數
        int sum = 0;
        for (int i = 0; i < 11; i++) { //
            for (int j = 0; j < 11; j++) {
                if(array1[i][j]!=0) {
                    sum++;
                }
            }
        }
        System.out.println("有效值的個數："+sum);

        // 2. 建立一個稀疏陣列的陣列
        int[][] array2 = new int[sum+1][3]; // new int[行][列]

        array2[0][0] = 11; // 行
        array2[0][1] = 11; // 列
        array2[0][2] = sum; // 值

        // 遍歷二維陣列，將非0的值，存到稀疏陣列中
        int count = 0;
        for (int i = 0; i < array1.length; i++) { // 11*11的棋盤
            for (int j = 0; j < array1[i].length; j++) { // 有值的地方
                if(array1[i][j]!=0) {
                    count++;
                    array2[count][0] = i; // 從第count行開始，儲存橫坐標
                    array2[count][1] = j; // 從第count行開始，儲存縱坐標
                    array2[count][2] = array1[i][j]; //
                }
            }
        }

        // 輸出稀疏陣列
        System.out.println("輸出稀疏陣列");

        for (int i = 0; i < array2.length; i++) {
            System.out.println(array2[i][0]+"\t" // 只有3列
                    +array2[i][1]+"\t"
                    +array2[i][2]+"\t");
        }

        System.out.println("還原稀疏陣列");
        // 1. 讀取稀疏陣列的值
        int[][] array3  = new int[array2[0][0]][array2[0][1]];

        // 2. 給其中的元素還原值
        for (int i = 1; i < array2.length; i++) {
            array3[array2[i][0]][array2[i][1]] = array2[i][2];
        }

        // 3. 輸出
        System.out.println("還原的陣列");

        for (int[] ints : array3) {
            for (int anInt : ints) {
                System.out.print(anInt+"\t");
            }
            System.out.println();
        }
    }
}

# 4. 隨機交換

public class Demo01 {

        }

    }
}

# 12. 查找

# 1. 基本查找

根據元素查找出該元素在陣列中第一次出現的索引值

public class ArrayDemo2 {

    public static void main(String[] args) {

        int[] arr = {1, 5, 7, 4, 5, 2, 4, 6, 2, 5, 3};
        int index = getIndexEle(arr, 5);
        System.out.println(index);

    }
    private static int getIndexEle(int[] arr, int ele) {
        for (int i = 0; i < arr.length; i++) {
            if(arr[i]==ele) {
                return i;
            }
        }

        return -1; // 找不到就返回-1
    }
}

# 2. 二分查找

前提是陣列元素必須按照順序，如果沒有順序，就得使用基本查找

public class ArrayDemo3 {

    public static void main(String[] args) {

        int[] arr = {10, 20, 30, 40, 50, 60, 70, 80};

        int index = getIndexByEle(arr, 50);
        System.out.println("索引"+index);

    }

    private static int getIndexByEle(int[] arr, int ele) {
        int min = 0;
        int max = arr.length - 1;
        int center = (min+max) / 2;

        while (min <= max) {
            if (ele == arr[center]) {
                return ele;
            } else if (ele > arr[center]) {
                min = center + 1;
            } else if (ele < arr[center]) {
                max = center - 1;
            }
        center = (min + max) / 2;

        }

        return -1;
    }
}

# 7. 陣列排序

# 1. 冒泡排序

陣列元素兩兩比較，交換位置，大元素往後放，經過一輪比較後，最大的元素就會出現在最大索引處

public class ArrayDemo5 {
    public static void main(String[] args) {

        int[] arr = {1,45,436,57,253,1,35};

        for (int i = 0; i < arr.length-1; i++) {
            for (int j = 0; j < arr.length-1-i; j++) {
                if(arr[j]>arr[j+1]) {
                    int t = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+i] = t;
                }
            }
        }
        System.out.println(Arrays.toString(arr));
    }
}

# 2. 選擇排序

從0索引處開始，一次和後面的元素進行比較，小的元素往前放，經過一輪比較後，最小的元素就出現在最小索引處

public class ArrayDemo6 {

    public static void main(String[] args) {

        int[] arr = {24, 69, 80, 57, 13};

        for (int index = 0; index < arr.length-1; index++) {
            for (int i = 1; i < arr.length; i++) {
                if(arr[index]>arr[i]) {
                    int temp = arr[index];
                    arr[index] = arr[i];
                    arr[i] = temp;
                }
            }
        }
    }
}

# 3. 直接插入排序

將一個紀錄插入到一個長度為m的有序表中，始知仍保持有序

public class ArrayDemo4 {
    public static void main(String[] args) {
        int[] arr = {3, 2, 1, 0, 10, 20, 30, 7, 8};

        // 1. 外層循環定義輪次
        for (int i = 1; i < arr.length; i++) {
            int j = i;
            while(j > 0 && arr[j] < arr[j-1]) {
                int t = arr[j];
                arr[j] = arr[j-1];
                arr[j-1] = t;
                j--;
            }
        }
        System.out.println(Arrays.toString(arr)); // [0, 1, 2, 3, 7, 8, 10, 20, 30]


        // 2. 內層循環
        for (int i = 1; i < arr.length; i++) {
            for (int j = i; j >0; j--) {
                if(arr[j] < arr[j-1]) {
                    int t = arr[j];
                    arr[j] = arr[j-1];
                    arr[j-1] = t;
                }
            }
        }
        System.out.println(Arrays.toString(arr)); // [0, 1, 2, 3, 7, 8, 10, 20, 30]
    }
}
