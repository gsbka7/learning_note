# 02 【安裝】

# 1.開發環境

開發 Laravel 專案前，需先建置以下開發環境。

1.Xampp or MAMP

2.Visual Studio Code

3.Composer

4.Node.js

5.HeidiSQL

# 1.1 Xampp or MAMP

使用 Xampp 與 MAMP 可在本機端建置網頁伺服器。

Xampp：進入官網open in new window

MAMP：進入官網open in new window

# 1.2 Visual Studio Code

Visual Studio Code，跨平台的原始碼文字編輯器

進入官網open in new window

# 1.3 Composer

Composer，PHP套件管理工具。

進入官網open in new window

# 1.4 Node.js

NPM 是Node.js 的套件管理工具，Node.js 中內建 NPM，需透過安裝 Node.js 的方式去安裝 NPM。

進入官網open in new window

# 1.5 HeidiSQL

HeidiSQL為資料庫可視化管理工具。

進入官網open in new window

# 2.建立 Laravel 專案

# 2.1 安裝 Laravel

開啟 VS Code ，啟動 TERMINAL（ctrl + ~），輸入指令。

composer global require laravel/installer

# 2. 建立專案

於根目錄建立專案。

composer create-project --prefer-dist laravel/laravel:^7.0 Project Name

Project Name：專案名稱，自行命名。

# 2.3 安裝 Composer 與 NPM

1.從根目錄進入專案存放之資料夾。

cd Project Name

2.安裝 Composer

composer install

3.安裝 NPM

NPM install

# 2.4 修改配置

1「.env檔」

設置與資料庫連接之配置。

DB_PORT=3306

需與 Xampp 的連接埠相同

DB_DATABASE=Database

需與 HeidiSQL 中建立的資料庫名稱相同。

# 2.5 測試連接

php artisan serve

連接成功，會出現 http://127.0.0.1:8000 ，於該網址進入專案頁面。
