# 01 【 000WebHost 】

支援 PHP / MySQL

進入 000WebHostopen in new window

# 02 【 部屬專案 】

將 Laravel 專案 部屬至 000WebHost

# 1. 建立網站

# 2. FTP 上傳

使用 FTP 軟體將專案檔案上傳至虛擬主機

# 1. 安裝 FileZilla

進入 FileZillaopen in new window

# 2. 連線至主機

000webhost -> Website Settings -> FTP Detailed Information

使用主機名、帳號、密碼、連線埠登入主機

登入後會取得主機目錄

# 3. 檔案上傳

將資料夾中除了 node_modules 和 public 以外的所有檔案上傳至目錄最上層

public 裡的檔案上傳至 public_html

# 4. 設定連接資料庫

# 4.1 新增資料庫

Tools -> Database Manager -> New Database

# 4.2 匯入資料庫

無須匯入可省略

Manage -> PhpMyAdmin

# 4.2 資料庫連線設定

開啟 .env 檔，將設定寫入並保存。
