# 01 【 介紹與安裝配置 】

# 1. 資料庫 與 資料庫管理系統

# 1. 資料庫：DataBase，簡稱DB

・ 儲存資料的倉庫，儲存了具有特定格式資料的文件

# 2. 資料庫管理系統：DataBaseManegement，簡稱DBMS

・ 可以對資料庫裡的資料進行增刪改查

・ 常見資料庫管理系統：MySQL、Oracle

# 3. SQL

・ 結構化查詢語言

・ 通過編寫SQL語句，DMBS執行SQL語句，執行資料庫中資料的增刪查改

・ SQL 是一套標準，SQL語句在MySQL中可以使用，Oracle也可以使用。

# 4. 關係

DBMS -> 執行 -> SQL -> 操作 -> DB

# 2. 安裝

# 1. 下載

MySQL Community Downloadsopen in new window

# 2. 安裝模式

# 1. Developer Default 模式

此模式會安裝開發人員需要的常用組件，在安裝這些組件時需要對應的環境依賴，需要安裝設置好依賴的環境後再進行下一步安裝

# 2. Custom 模式

自定義安裝

# 3. 配置

# 1. Type and Networking

連線埠預設：3306

# 2. Accounts and Roles

・ 設定 root 密碼

・ 設定使用者帳號密碼

# 4. MySQL服務啟動與停止

# 1. 本機 -> 管理 -> 服務與應用程式 -> 服務 -> MySQL -> 右鍵(啟動)

# 2. 命令提示字元

・ 以系統管理員身分開啟cmd

・ 啟動：執行 net start mysql

・ 停止：執行 net stop mysql

# 3. MySQL管理工具

當完成資料庫的安裝之後，MySQL是以服務的形式運作在windows/linux系統，使用者是通過DBMS工具來對MySQL進行操作的。當安裝MySQL會默認安裝 MySQL Commcand line Client，此工具是命令行形式的工具，通常還會安裝可是化的DBMS工具

# 1. MySQL Commcand line Client 使用

  1. 打開 MySQL Commcand line Client

・ 開啟 cmd -> cd C:\ProgramData\Microsoft\Windows\Start Menu\Programs\MySQL\MySQL Server 8.0 -> MySQL 8.0 Command Line Client

・ 開啟 cmd -> 進入到bin目錄的路徑下 -> 輸入mysql -u root -p。

  1. 連接MySQL：

・ 輸入密碼 root password

・ 密碼錯誤會退出cmd

  1. 退出MySQL Commcand line Client

・ 輸入 exit
